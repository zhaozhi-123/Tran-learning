# 大众点评风格网页应用

这是一个功能完整、用户体验优良的类大众点评风格网页应用，实现了核心商业功能与用户交互体验。

## 核心功能模块

1. **商家信息展示**
   - 商家列表页：响应式卡片布局，包含商家图片、名称、评分、标签、地址等信息
   - 商家详情页：完整展示商家信息、用户评价、评分等
   - 分类导航：支持按类别浏览商家

2. **用户评价系统**
   - 查看用户评价
   - 提交评价和评分
   - 5星评分系统

3. **用户注册与登录**
   - 注册新用户
   - 用户登录
   - 登录状态管理

4. **商家搜索与筛选**
   - 关键词搜索
   - 距离、评分、排序筛选

5. **响应式设计**
   - 桌面端、平板端、移动端适配
   - 流畅的用户体验

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (原生)
- **后端**：Node.js + Express + MongoDB
- **数据库**：MongoDB
- **响应式设计**：媒体查询 + 弹性布局

## 运行指南

### 1. 安装依赖

```bash
npm install
```

### 2. 启动数据库

确保MongoDB服务已经启动，默认连接到 `mongodb://localhost:27017/dianping`

### 3. 启动服务器

```bash
npm start
```

或使用开发模式（自动重启）：

```bash
npm run dev
```

### 4. 访问应用

在浏览器中打开 `http://localhost:3000`

## 功能演示

1. **浏览商家**：在首页查看商家列表，点击商家卡片查看详情
2. **登录/注册**：点击右上角的登录或注册按钮
3. **写评价**：在商家详情页登录后可以提交评价和评分
4. **筛选商家**：使用顶部的分类导航和筛选栏
5. **搜索商家**：在顶部搜索栏输入关键词搜索

## 响应式设计

- **桌面端**：多列网格布局，完整展示所有信息
- **平板端**：双列布局，优化空间利用
- **移动端**：单列布局，简化导航和交互

## 技术特点

- **模拟数据**：内置模拟商家数据，确保在API调用失败时也能展示内容
- **模块化代码**：清晰的代码结构和完整的注释
- **用户体验**：流畅的交互效果和直观的界面设计
- **可扩展性**：易于添加新功能和扩展现有功能
