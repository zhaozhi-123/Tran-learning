冒泡排序原理

步骤 1: 比较相邻元素

从数组的第一个元素开始，依次比较相邻的两个元素。

步骤 2: 交换元素

如果前一个元素大于后一个元素，则交换它们的位置。

步骤 3: 重复过程

对每一对相邻元素执行相同的操作，直到到达数组末尾。这样，最大的元素会"冒泡"到数组的末尾。

步骤 4: 缩小范围

针对所有元素重复以上步骤，除了最后一个已经排好序的元素。

步骤 5: 完成排序

当没有任何一对数字需要交换时，排序完成。

冒泡排序代码

function bubbleSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        // 标记是否发生交换
        let swapped = false;
        for (let j = 0; j < n - i - 1; j++) {
            // 比较相邻元素
            if (arr[j] > arr[j + 1]) {
                // 交换元素
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
                swapped = true;
            }
        }
        // 如果没有发生交换，说明数组已经有序
        if (!swapped) break;
    }
    return arr;
}

算法分析

时间复杂度

  • 最好情况: O(n) - 当数组已经有序时
  • 平均情况: O(n²)
  • 最坏情况: O(n²) - 当数组逆序时

空间复杂度

O(1) - 只需要常数级的额外空间

稳定性

稳定 - 相同值的元素在排序后相对位置不变
